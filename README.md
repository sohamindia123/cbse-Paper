# CBSE Question Paper Generator

A simple Flask web application that generates CBSE-style question papers for Classes 9â€“12.

## Installation

```bash
# Clone the repository
git clone https://github.com/sohamindia123/cbsepaper.git
cd cbsepaper

# Create and activate a virtual environment
python -m venv venv
# macOS/Linux
source venv/bin/activate
# Windows
venv\Scripts\activate

# Install dependencies
pip install -r requirements.txt
```

## Usage

```bash
# Start the application
python app.py
```

Open your browser and go to `http://127.0.0.1:5000/` to use the generator.
